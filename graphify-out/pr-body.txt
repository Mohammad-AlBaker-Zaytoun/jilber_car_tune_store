## Summary

Adds a graphify knowledge graph built from the full Next.js project source.

- 261 nodes (229 from AST structural extraction + 32 from semantic doc analysis)
- 382 edges covering imports, references, calls, and semantic relationships
- 6 named communities detected by Louvain clustering:
  - Cart & Platform Core
  - Admin & Order Management
  - Next.js Framework & Rendering
  - Auth & Session System
  - User Domain Types
  - Contact & WhatsApp

### God nodes (highest degree / most connected)
- JILBER Performance Engineering Platform (degree 16) — README hub
- Auth Flow (degree 6) — bridges JWT, session, and admin auth
- Quote Status Lifecycle (degree 6) — cross-cuts admin + account views

### Surprising connections
- SessionUser → UserRole (auth.ts ↔ types/admin.ts) — cross-file type coupling
- StoredUser → UserRole (users.dev.ts ↔ types/admin.ts) — dev store shares admin type
- Next.js Breaking Changes Warning ↔ Next.js 16 App Router (AGENTS.md ↔ README.md)
